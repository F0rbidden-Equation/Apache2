## Liste de Commandes pour Apache2 sur Linux
Demmarrer Apache2
```bash
     sudo service apache2 start
```
Arreter Apache2
```bash
     sudo service apache2 stop
```

Redemarrer Apache2
```bash
     sudo service apache2 start
```

Verifier l'etat du webserver apache2
```bash
     sudo service apache2 status 
```
